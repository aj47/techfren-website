TribunalBench tests AI agents' ability to maintain logical reasoning as their context window fills up. Unlike traditional benchmarks, this is a fully deterministic logic puzzle with exactly one valid solution.

The challenge: classify 30 witnesses as either ‚öîÔ∏è Knights (always tell truth) or üó°Ô∏è Knaves (always lie) based on their statements about each other.

Two difficulty levels:
‚Ä¢ Normal: 3 statements per witness, ~2,000 tokens/query ‚Üí ~45K-60K total tokens
‚Ä¢ Hard: 5 statements per witness, ~4,500 tokens/query ‚Üí ~135K+ total tokens

üîß DESIGN PRINCIPLES

  • Deterministic: No luck or randomness. The puzzle has exactly one logically valid solution derived through pure deduction.
  • Cheat-Proof: The solution is never stored. Validation checks logical consistency only ‚Äî if a Knight's statements are all true and a Knave's are all false, you win.
  • Context-Heavy: Each character query returns 2,000+ tokens of verbose text (backgrounds, statements, cross-references, scenario analysis). Solving requires tracking information across a massive context.
  • Bot-Friendly: Exposes a clean API: queryCharacter(), classify(), checkConsistency(), submitAnswer()

üéØ WHY IT MATTERS

Current AI benchmarks often test knowledge recall or single-turn reasoning. But real-world agent tasks require sustained logical coherence across many interactions while context accumulates.

This benchmark reveals: Do agents "forget" earlier deductions as context grows? Can they maintain a mental model of 30 interconnected entities? Do they fall into circular reasoning or contradictions?

The hypothesis: Performance will degrade as context fills ‚Äî and different models will degrade at different rates. This benchmark makes that visible.
